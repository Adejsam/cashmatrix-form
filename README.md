# Cashmatrix Account Opening Form

A clean and professional account opening form for **Cashmatrix**. This project is built using semantic HTML5, modern CSS3, and is optimized for web deployment.

## Features

- **Responsive Design**: Adapts beautifully to different screen sizes.
- **Form Validation**: Simple and intuitive form fields for account registration.
- **Sleek Aesthetics**: Modern typography and a clean layout using custom CSS.
- **SEO Optimized**: Standard meta tags and structured headings.

## Technologies Used

- **HTML5**: For semantic structure.
- **CSS3**: For styling and responsive layout.
- **Node.js**: For local development and serving the files.

## Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS version recommended)

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repository-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the local development server:

```bash
npm start
``` 

The server will typically run at `http://localhost:3000` (or the port specified by `serve`).

## Project Structure

- `index.html`: The main entry point for the form.
- `styles.css`: Custom CSS styles for the application.
- `cashmatrix logo.png`: Project assets.
- `.gitignore`: Configured to exclude `node_modules` and other unnecessary files.
- `package.json`: Project metadata and dependencies.
